# Calculate Number (Sum and Multiplication) in Android App

![alt text](<WhatsApp Image 2025-12-15 at 15.07.04_3a5413dd.jpg>)

This Android application allows users to input two numbers and calculate their sum and multiplication. The app features a simple user interface with input fields for the numbers and buttons to perform the calculations.

## Features
- Input fields for two numbers
- Buttons to calculate the sum and multiplication
- Display results on the screen

## Usage
1. Enter two numbers in the provided input fields.
2. Click the "Calculate Sum" button to see the sum of the two numbers.
3. Click the "Calculate Multiplication" button to see the product of the two numbers.

## Installation
1. Clone the repository to your local machine.
2. Open the project in Android Studio.
3. Build and run the application on an Android device or emulator.
